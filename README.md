# angle-vue-preset

The vue-cli preset of angle-FE team

## How To Use

> use preset from GitHub repo

`vue create -p youngbeen/angle-vue-preset my-project`

## Content

### H5 template

* Vue router
* ESLint support
* Less support
* px2rem support
* [optional] - encryption module
* Testing/Production environment support
* Lint report generating
* gz compression
* Bundle analyzer

### PC template

* Vue router
* ESLint support
* Sass support
* [optional] - encryption module
* Testing/Production environment support
* Lint report generating
* gz compression
* Bundle analyzer
